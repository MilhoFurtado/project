Instalando o APACHE pelo terminal.

  1. Abra o terminal: Em muitas distribuições Linux, você pode abrir o terminal pressionando Ctrl + Alt + T. Se não for esse o caso, procure o terminal em seu menu de aplicativos.

  2. Atualize o sistema: Antes de instalar qualquer pacote, é uma boa prática atualizar o sistema.
    Digite o seguinte comando no terminal e pressione Enter:

    sudo apt update
    sudo apt upgrade

    Isso irá atualizar a lista de pacotes disponíveis e instalar as atualizações disponíveis no sistema.

  3. Instale o Apache: Agora, você pode instalar o Apache usando o comando apt. Digite o seguinte comando no terminal e pressione Enter:

    sudo apt install apache2

    O sistema irá solicitar sua senha de administrador (sudo) para confirmar a instalação. Digite sua senha e pressione Enter. Em seguida, o sistema irá baixar e instalar o Apache.

  4. Inicie o Apache: Após a instalação, o Apache é iniciado automaticamente. Você pode verificar se o serviço está em execução digitando o seguinte comando no terminal:

    sudo service apache2 status

    Se o Apache estiver em execução corretamente, você verá uma mensagem indicando que o serviço está ativo.

Agora você tem o Apache instalado em seu sistema Linux e pode começar a hospedar sites ou aplicativos da web. Os arquivos do site são armazenados no diretório "/var/www/html", onde você pode adicionar seus próprios arquivos HTML, CSS, JavaScript e outros recursos para exibi-los no navegador.
